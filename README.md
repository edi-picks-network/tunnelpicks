# Tunnel Picks

A modern VPN comparison and review website built with Next.js 15, Tailwind CSS v3.4, and TypeScript. Features a red color scheme.

## Color Palette (Red)

- **Primary bg**: `#0A0A1A` (deep black)
- **Secondary**: `#111128` (dark)
- **Tertiary**: `#1A1A3E`
- **Border**: `#2D2D6B` (muted)
- **Active border**: `#B91C1C` (red-700)
- **Accent**: `#DC2626` (red-600)
- **Accent glow**: `#EF4444` (red-500)
- **Text primary**: `#F0F0FF` (near-white)
- **Text secondary**: `#A5B4FC` 
- **Text muted**: `#6B7280` (gray-500)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS v3.4
- TypeScript
- Framer Motion
- Lucide Icons
- Space Grotesk font
