# tunnelpicks

**Tunnel Picks** — a VPN comparison and review site with a red color scheme.

Built with Next.js 15, Tailwind CSS v3.4, and TypeScript.

Color palette (Red):
- Primary bg: #0A0A1A (deep black)
- Secondary: #111128 (dark)
- Tertiary: #1A1A3E
- Border: #2D2D6B
- Active border: #B91C1C (red-700)
- Accent: #DC2626 (red-600)
- Accent glow: #EF4444 (red-500)
- Text primary: #F0F0FF (near-white)
- Text secondary: #A5B4FC
- Text muted: #6B7280 (gray-500)
- Gold/Star: #F59E0B
